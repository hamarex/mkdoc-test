# マークダウンからWEBページ(wiki)へ (Doc as a Code)

!!! info "エグゼクティブ・サマリー"
    本プラットフォームは、全社の技術資産・ノウハウを「コード」として管理することで、**情報の陳腐化を防ぎ、開示スピードを300%向上**させることを目的としています。

## 見せる用のページです。

=== "1. 情報のサイロ化"
    各部署に散らばったWordやExcelを、単一の検索可能なポータルへ集約します。
=== "2. 更新コスト"
    手動のPDF変換やアップロード作業を自動化。エンジニアが「書く」だけで全社に公開されます。
=== "3. ガバナンス"
    「誰が、いつ、なぜ変更したか」をGit履歴で100%追跡可能にします。
