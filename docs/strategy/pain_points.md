# 従来手法とのコスト比較分析

ドキュメント作成・管理にかかる年間工数の試算です。

| 項目 | 従来のOffice管理 | Doc as a Code | 改善率 |
| :--- | :--- | :--- | :--- |
| 検索・参照時間 | 120時間/年 | 20時間/年 | **83%削減** |
| フォーマット調整 | 40時間/年 | 0時間/年 | **100%削減** |
| 承認・公開フロー | 1週間 | 1時間以内 | **98%短縮** |

!!! tip "コストの再投資"
    削減された時間は、ドキュメントの「整形」ではなく「内容の充実」に充てることが可能になります。
