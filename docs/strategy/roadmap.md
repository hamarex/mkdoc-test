# IT投資・ドキュメント基盤ロードマップ

本プロジェクトは、単なるドキュメントツールの導入ではなく、**「企業の知的財産をデジタル資産化する」**ための3カ年計画です。

## ロードマップ

### Short-term: 基盤確立 (Year 1)
*   **Doc as a Code環境の構築**: Markdown + Gitによる執筆環境の全社標準化。
*   **技術ドキュメントの移行**: PDFや社内Wikiに散在する重要情報の集約。
*   **CI/CD自動化**: 修正から公開までの完全自動化プロセスの稼働。

### Mid-term: ナレッジエコシステム形成 (Year 2)
*   **AIチャットボット連携**: 蓄積されたMarkdownデータを学習させ、社内規定に即答するAIを導入。
*   **全部署への展開**: 人事・法務規定など、非エンジニア部門への適用拡大。
*   **分析ダッシュボード**: どのドキュメントが頻繁に参照されているかを可視化。

### Long-term: 知的資産の完全同期 (Year 3)
*   **リアルタイム・ガバナンス**: 仕様変更とドキュメント更新が完全に同期する体制の確立。
*   **外部公開プラットフォーム化**: 顧客向けマニュアルやAPIリファレンスを本基盤から直接配信。
